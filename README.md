This is an led set for 5mm leds and esp32 and humidity sensor on the top extrusion bar of an ender 3 pro and on the top of the side vertical psu



list of materials

\-1 3 pin switch 

\-15 5mm leds (i chose white)

\-15 220Ω resistors

\-lots of jumpers (mine are variable size)

\-1 Arduino uno r3 (mine is usb c a 3d printed case will have other port options in the future)

\-1 dht11 humidity sensor

\-2 10 pin 7 seg led displays

-2 170 pin breadboards

-optional 3d printed hardware

note: all parts are made with pla filament and all hardware is trying to be completely 3d printed.

ALL code is in c++ using the arduino ide with the uno boards set and the libraries SevSeg and adafruit dht library.
