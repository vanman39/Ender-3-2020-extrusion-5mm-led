This is an led set for 5mm leds and esp32 and humidity sensor.



list of materials

\-1 3 pin switch 

\-15 5mm leds (i chose white)

\-15 220Ω resistors

\-lots of jumpers (mine are variable size)

\-1 Arduino uno r3

\-1 dht11 humidity sensor

\-2 7 seg led displays

\2 170 pin breadboards

\optional 3d printed hardware
